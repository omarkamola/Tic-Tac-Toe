# Tic-Tac-Toe Game (Python Version)


This is a simple **Tic-Tac-Toe** game built using Python as part of my learning journey in the *Cisco Python 1 Course*.  
The game is played between the computer (**X**) and the user (**O**) directly in the console.

---

##  Features
- The computer always starts and places 'X' in the center.
- The user selects moves by entering a number from **1 to 9**.
- Prevents invalid or duplicate moves.
- Random move generation for the computer.
- Detects win, loss, or draw conditions.
- Clean and readable board display.

---

##  Technologies Used
- **Python 3**
- `random` module
- Loops, lists, conditions, and functions
